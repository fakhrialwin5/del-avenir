# ── Drone video input folder ──────────────────────────────────────────────
#
# Drop your drone / landscape footage here. The component automatically picks
# it up on the next `npm run dev` / `npm run build` (a manifest is regenerated
# right before those commands run).
#
# Rules:
#   • Use ONE video at a time (the first supported file wins).
#   • Supported: .mp4  .webm  .mov  .m4v  .ogv
#   • Prefer .mp4 (best browser support). Keep it reasonably compressed.
#   • Wide / landscape footage works best for the drone pan effect.
#
# Then run the app — no code change needed.
